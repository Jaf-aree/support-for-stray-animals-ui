# Support for Stray Animals - UI/UX Prototype

A clean UI/UX prototype designed to support stray animals, streamline the adoption process, manage donations, and report animals in need.

## Key Pages & Features
* **Homepage:** The main landing page showcasing success statistics, community news, and platform navigation.
* **Report Page:** A quick and straightforward form for citizens to report injured or vulnerable stray animals.
* **Adopt Page:** An intuitive interface containing lists, filters, and profile details for animals ready for adoption.
* **Donate Page:** A secure portal dedicated to collecting funds for animal food, shelter, and medical needs.
* **Thank You Page:** A warm post-action screen confirming successful contributions or actions.

## Design Approach
* **Component-Based:** Developed inside Figma utilizing clean frames, components, and auto-layouts for modularity.
* **User-Centric:** Uses a welcoming color scheme and accessible layout to drive community trust and high engagement.
